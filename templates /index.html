JS-Free Marketplace Static Demo

About This Application

This is a Python Flask-based marketplace for cryptocurrency-related products, designed to function without client-side JavaScript. This static version demonstrates the UI templates but lacks backend functionality.

Application Features

  • User authentication (login/register)
  • Product listings and details
  • Shopping cart functionality
  • Checkout process
  • Order management
  • Admin dashboard
  • Vendor dashboard
  • Cryptocurrency payments
  • Escrow system
  • Activity logging

Backend Not Available

The Python Flask backend that powers this marketplace is not available in this deployment. To fully deploy this application, consider using a platform that supports Python backends like:

  • Heroku
  • AWS Elastic Beanstalk
  • Google App Engine
  • DigitalOcean App Platform
  • Render

Deployment Instructions

Detailed deployment instructions can be found in the README.md file in the project repository.

Key steps include:

  1. Choose a platform that supports Python Flask applications
  2. Set up a PostgreSQL database service
  3. Configure environment variables
  4. Deploy the Flask application
